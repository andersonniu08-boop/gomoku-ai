# Gomoku AI

AlphaZero-style Gomoku (15×15, 5-in-a-row) engine combining Monte Carlo Tree Search with deep neural networks.

**Status: Work in progress — active development.**

## Overview

This engine plays Gomoku at superhuman strength by fusing classical tree search with learned neural intuition, following the AlphaZero paradigm. Built with PyTorch and NumPy for research and experimentation.

See [CLAUDE.md](CLAUDE.md) for architecture, design rules, and roadmap.
